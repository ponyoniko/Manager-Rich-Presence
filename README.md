# Manager Rich Presence | MRP

![MRP Logo](https://media.discordapp.net/attachments/1452446137712644127/1452446158914850826/icon.png?ex=6949d765&is=694885e5&hm=1e1208187edd096b73051a92beac8be3a1ab5ebbbec9eb2096a49c9d9a813744&=&format=webp&quality=lossless&width=550&height=550)

**Manager Rich Presence (MRP)** is a powerful and user-friendly Discord Rich Presence manager that allows full customization of your Discord activity. Designed for both beginners and advanced users, MRP gives you complete control over what your friends see on Discord.

---

## üåü Features

- **Full Rich Presence Control**: Customize every aspect of your Discord activity, including details, state, images, timestamps, and party size.  
- **Preset System (.mrp)**: Save, load, and share presets effortlessly with the custom `.mrp` format.  
- **Image Assets Support**: Use large and small images with custom hover text directly from Discord assets.  
- **Advanced Timestamps**: Display elapsed time, app start time, local time, or fully custom timestamps.  
- **Custom Buttons**: Add up to two clickable buttons linking to websites, Discord servers, or profiles.  
- **Language System**: Load translations dynamically from `.lrp` files. Add new languages without touching the code.  
- **Theme Customization**: Switch between dark and light mode, with more themes planned in future updates.  
- **User-Friendly Design**: Simple enough for beginners, powerful enough for advanced users.
